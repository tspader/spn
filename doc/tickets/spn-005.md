All CLI Commands

  Main Commands:
  1. init - Initialize a project
  2. add - Add a package
  3. build - Build the project
  4. print - Print compiler flags
  5. copy - Copy binary dependencies
  6. link - Link dependencies (alias for copy)
  7. update - Update a package
  8. list - List all packages
  9. which - Print cache directory path
  10. ls - Run ls on cache directory
  11. manifest - Print manifest source
  12. clean - Clean build/cache directories
  13. tool - Tool management subcommand

  Subcommands (under tool):
  1. tool install - Install a package's binaries
  2. tool uninstall - Uninstall a package's binaries
  3. tool run - Run a package's binary
  4. tool list - List installed tools
  5. tool upgrade - Upgrade a tool

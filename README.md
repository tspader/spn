# quickstart
Install:
```bash
curl https:
```
```bash
spn init
spn add sdl3
spn build
gcc main.c $(spn flags include) -lSDL3 -o hello
./hello
> "spn is a simple, stupid package manager for c inspired by uv and lazy.nvim"
```
